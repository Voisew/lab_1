# Лабораторная работа 1
## Текстовый редактор
## Описание
Данный проект представляет собой текстовый редактор с графическим интерфейсом пользователя (GUI), который поддерживает базовые функции редактирования текста и работы с файлом.
## Основные элементы интерфейса:
Главное меню программы <br>
Панель инструментов с кнопками быстрого доступа <br>
Область для ввода и редактирования текста <br>
Область отображения результатов работы языкового процессора (только для чтения) <br>
## Функции текстового редактора
Текстовый редактор поддерживает все основные функции работы с документами: создание новых файлов, открытие существующих и их сохранение в различных форматах. Пользователь может отменять и повторять действия, а также выполнять стандартные операции с текстом: копирование, вырезание и вставку фрагментов. Все операции доступны как через главное меню, так и с помощью панели инструментов и горячих клавиш.
## Главное меню программы
Действия, связанные с управлением файлами:<br>
Создать  – Создание нового файла/проекта.<br>
Открыть – Открытие существующего файла из памяти компьютера.<br>
Сохранить  – Сохранение текущего файла (если файл новый, открывает диалог "Сохранить как").<br>
Сохранить как…  – Сохранение файла под новым именем или в другом формате.<br>
Выход  – Закрытие программы (предлагает сохранить изменения, если они есть).<br>
Правка:<br>
Операции с текстом/объектами:<br>
Отменить  – Отмена последнего действия.<br>
Повторить  – Возврат отменённого действия.<br>
Вырезать  – Удаление выделенного фрагмента с помещением в буфер обмена.<br>
Копировать  – Копирование выделенного фрагмента в буфер обмена.<br>
Вставить  – Вставка содержимого буфера обмена.<br>
Удалить  – Удаление выделенного фрагмента (без сохранения в буфере).<br>
Выделить все  – Выделение всего содержимого документа/области.<br>
## Вызов справки
Для быстрого доступа к руководству пользователя:<br>
Нажмите кнопку «Справка» () на панели инструментов.<br>
Используйте пункт меню «Справка → Руководство пользователя».<br>
Руководство содержит:<br>
- Подробное описание всех функций редактора<br>
- Пошаговые инструкции по работе с программой<br>
- Ответы на частые вопросы (FAQ)<br>

Чтобы узнать информацию о версии и разработчиках:<br>
Нажмите «Справка → О программе».<br>
Или воспользуйтесь кнопкой «О программе» () на панели инструментов.<br>

# Лабораторная работа 6
## Задачи
1.Построить РВ, описывающее шестнадцатеричный код (HEX) 
цвета (длиной 6).
2.Построить РВ, описывающее номера карт, принадлежащих
платежной системе Visa.
3.Построить РВ для поиска HSL-кода цвета. Пример: hsl(120,
50%, 40%) 
## Регулярные выражения
Задача 1 <br>
HEX: начинается с #, далее ровно 6 символов 0-9 или a-f (в любом регистре) <br>
```
(@"#([0-9a-fA-F]{6})
```
Задача 2 <br>
Visa: начинается с 4, далее 12–18 цифр (обычно 16) <br>
```
(@"\b4[0-9]{12}(?:[0-9]{3})?\b) 
```
Задача 3 <br>
  hsl(число, процент, процент), возможны пробелы <br>
  ```
 (@"hsl\(\s*(\d{1,3})\s*,\s*(\d{1,3})%\s*,\s*(\d{1,3})%\s*\)" 
```
